# Áp dụng thuật toán để tối ưu việc vận chuyển cho các bài toán Logistic

Dự án này áp dụng thuật toán **SHO** (Tối ưu hóa đàn linh cẩu) và các kỹ thuật: **Swap Operation, Swap Sequence, PD_Shift, PD_Exchange, PD_Rearrange**. Những kỹ thuật này cũng được dùng cho các thuật toán **GWO** và **WOA** để đề xuất cách tiếp cận mới, cải thiện lập lộ trình tuyến xe phù hợp với yêu cầu thực tế. ***Mục tiêu là rời rạc hóa các giải thuật meta-heuristic để tìm lời giải tối ưu cho bài toán lập lộ trình tuyến xe trong không gian bài toán lớn.***
